# Galaxy-TEDX

A live interactive galaxy visualization for TED-style events, featuring dynamic idea nodes and mesh connections.

## Features
- Interactive node-based idea visualization
- Real-time mesh networking
- Particle effects and animations
- Live event statistics display

## Getting Started
Open `index.html` in your web browser to view the interactive visualization.

## Technologies
- HTML5 Canvas
- CSS3 Animations
- JavaScript
- Google Fonts (Bebas Neue, Space Mono, Space Grotesk)
